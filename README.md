## Descrição do projeto
Este projeto tem como objetivo visualizar o balanceamento que ocorre em árvores AVL através das operações de rotação.

## Guia de instalação

### Dependências do projeto

**Linguagem**: Typescript<br>
**Framework**: React, Node v22

### Como executar o projeto

```
git clone https://github.com/EDAII/ArvoresBalanceadas_AVLVisualizer.git
```

```
yarn install
```

```
yarn run dev
```
